walkmod:apply

Full name:

org.walkmod.maven.plugins:walkmod-maven-plugin:1.0.3:apply

Description:

Modifies the source code according the applied coding style conventions

Attributes:

  • Binds by default to the lifecycle phase: generate-sources.

Optional Parameters

Name Type Since Description
chains String - Chains to execute
Default value is: ${walkmod.chains}.
User property is: chains.
configFile File - Configuration file
User property is: configFile.
excludes String[] - Specific list of excluded files
includes String[] - Specific list of included files
offline boolean - If dependencies are resolved off-line
Default value is: false.
User property is: offline.
path String - Specific source file directory
User property is: path.
printError boolean - If prints errors
Default value is: false.
User property is: printError.
properties String - Dynamic parameters to set
Default value is: ${walkmod.properties}.
User property is: properties.
skipWalkmod boolean - If skip walkmod
User property is: skipWalkmod.
verbose boolean - If is executed in verbose mode.
Default value is: true.
User property is: verbose.
